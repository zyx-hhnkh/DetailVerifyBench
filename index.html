Hallucination Detection Benchmark

A comprehensive evaluation framework for detecting hallucinations in Vision-Language Models at the token level

GitHub Repo Leaderboard Dataset

About

This benchmark measures how accurately Vision-Language Models (VLMs) can identify false or incorrect descriptions in image captions. Evaluations are performed at the token level, providing fine-grained hallucination detection metrics including Precision, Recall, F1, and IoU.

🔍

Token-Level Metrics

Precision, Recall, F1, and IoU computed at both token and span levels for fine-grained evaluation.

🤖

Multi-Model Support

Evaluate Qwen3-VL, GPT-5.x, Gemini-3.x, Claude Opus, Step-3-VL, Kimi, Seed, and more.

📊

4 Hallucination Categories

Counterfactual, OCR, Spatial Relation, and Other — covering diverse hallucination types.

⚙️

Multiple Backends

vLLM batch inference, Transformers, OpenAI API, Google Gemini API, and Anthropic API.

Getting Started

Installation
# Clone the repository
git clone https://github.com/zyx-hhnkh/Hallucination-Detection-Benchmark.git
cd Hallucination-Detection-Benchmark

# Install dependencies
pip install -r requirements.txt

# Download dataset from ModelScope
pip install modelscope
modelscope download --dataset 'TwinkleMoon/Hallucination-Benchmark' --include 'test/*' --local_dir ./ms_data
Run Evaluation
python run_pipeline.py \
    --mode offline \
    --image_dir ./ms_data/test \
    --input_json ./ms_data/test/test_gt_withtag.jsonl \
    --model_select "Qwen3-VL-8B" \
    --model_path "/path/to/model" \
    --exp_name "my_experiment" \
    --work_dir ./ \
    --use_think
